Shelter App

Описание проекта:
Приложение предназначено для учета животных в приюте и управления заявками на их усыновление. Пользователи могут просматривать животных, отправлять заявки на усыновление, а администраторы — управлять животными и обрабатывать заявки.

Функции программы:

Регистрация и аутентификация пользователей.

Просмотр списка животных с деталями.

Отправка заявки на усыновление животного.

Администрирование: добавление, редактирование, удаление животных.

Обработка заявок на усыновление (подтверждение, отклонение).

Фильтрация животных по статусу.

Просмотр статистики по породам животных.

Инструкция по установке:

Склонировать репозиторий:

git clone <URL_репозитория>
cd shelter_app


Создать виртуальное окружение:

python -m venv .venv
.venv\Scripts\activate  # Windows
source .venv/bin/activate  # Linux / macOS


Установить зависимости:

pip install -r requirements.txt


Создать базу данных:

from app import create_app
from app.extensions import db
app = create_app()
with app.app_context():
    db.create_all()


Запустить приложение:

python run.py
