# 🐾 Shelter App — Приют для животных

![Python](https://img.shields.io/badge/Python-3.8%2B-blue?logo=python)
![Flask](https://img.shields.io/badge/Flask-2.x-000000?logo=flask)
![License](https://img.shields.io/badge/License-MIT-green)

**Shelter App** — это веб-приложение для управления приютом животных. Оно помогает администраторам отслеживать животных и обрабатывать заявки на усыновление, а пользователям — находить нового пушистого друга и подавать заявки на его усыновление.

---

## 📌 Основные функции

- 🔐 **Регистрация и аутентификация** пользователей  
- 🐶 **Просмотр списка животных** с подробной информацией (фото, возраст, порода, статус и т.д.)  
- 📝 **Подача заявки на усыновление**  
- 👮 **Администрирование**:
  - Добавление, редактирование и удаление животных  
  - Обработка заявок (подтверждение / отклонение)  
- 🔍 **Фильтрация животных** по статусу (например: «в поиске дома», «уже усыновлен»)  
- 📊 **Статистика по породам** — наглядный обзор популярности пород в приюте  

---

## 🛠️ Технологии

- **Backend**: [Flask](https://flask.palletsprojects.com/)
- **Аутентификация**: Flask-Login
- **База данных**: Flask-SQLAlchemy (SQLite по умолчанию)
- **Шаблоны**: Jinja2
- **Тестирование**: pytest
